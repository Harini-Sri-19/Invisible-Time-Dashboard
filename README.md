# Assets

This folder is reserved for static assets referenced by the README and project documentation, primarily dashboard screenshots taken after running the app locally, e.g.:

- `screenshot_hero.png`
- `screenshot_overview.png`
- `screenshot_patterns.png`
- `screenshot_trends.png`
- `screenshot_data.png`

To generate these, run `streamlit run app.py`, navigate through each tab, and save screenshots into this folder using the filenames above so they match the references in `README.md`.
